# ITSM Ticket Audit Dashboard

## Overview
A spreadsheet-based ticket audit workflow designed to review service desk ticket quality, summarize audit results, identify recurring documentation or routing issues, and support daily operational reporting.

## Purpose
This project demonstrates how raw ITSM ticket exports can be converted into a structured audit workflow for technician-level review, documentation quality tracking, categorization checks, and process improvement.

## What It Evaluates
- Ticket documentation quality
- Categorization accuracy
- Routing and assignment quality
- Escalation readiness
- Resolution detail completeness
- Technician-level audit trends
- Recurring opportunities for improvement

## Public Version
This public version uses mock data and recreated examples only.

## Tools / Concepts
- Excel-style audit workflow
- ITSM ticket exports
- Ticket quality review
- Technician-level reporting
- Operational dashboards
